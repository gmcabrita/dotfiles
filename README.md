## Pre-install script setup

- Privacy & Security:
  - Enable "Full Disk Access" for Terminal, Ghostty, Zed
  - Enable "Developer Tools" for Terminal, Ghostty, Zed
    - https://nexte.st/docs/installation/macos/#gatekeeper
- Accessibility:
  - Increase Text size to 16pt for supported apps
- Display:
  - Disable "Automatically adjust brightness"
- Battery:
  - Disable "Slightly dim the display while on battery power"
  - Wake for network access: "Never"
- Notifications:
  - Allow notifications when the display is sleeping: No
- Keyboard:
  - Swap fn and ctrl
  - Press globe key to: "Do nothing"
  - Disable everything in Accessibility
  - Disable everything in Spotlight
  - Disable everything in Mission Control except:
    - Mission Control
    - Application windows
  - App Shortcuts
    - All Applications
      - Show Help Menu = `⇧ ⌘ /` (disabled?)
    - Notes.app
      - Note List Search... = `⌘ P`
  - Input Sources > Edit
    - Disable everything
- Trackpad:
  - Disable "Tap to click"
  - Disable "Look up & data detectors"
  - Disable all gestures
- Desktop & Dock
  - Window Tiling
    - Disable "Tiled windows have margins"
- Change DNS
  - 8.8.8.8
  - 8.8.4.4
  - 2001:4860:4860::8888
  - 2001:4860:4860::8844
- Wallpaper:
  - Custom color: #000000
- `./install.zsh`
